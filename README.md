# Shopping Cart

> A simple shopping cart with vue.js

## Requirements

- node js [5.1.x]
- npm [3.3.x]
- vue cli [1.4.x]

![alt tag](https://github.com/codekerala/vuejs-shoppingcart/raw/master/thumb.png)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
